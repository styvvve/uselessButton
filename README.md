# Nuclear Football

An interactive and humorous web project: a “nuclear launch button” that does absolutely nothing.  
Click it, watch the counter rise, and enjoy funny messages along the way.

## Features

- 3D round button with hover & click effects
- Click counter
- Humorous messages that change with the number of clicks
- Responsive design for all screen sizes
- Glassy/frosted background effect

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Click the big red button and enjoy
